## About

This repository serves as a personal archive of LeetCode solutions. Each solution includes:

- Problem description (brief)
- Solution code
- Time and space complexity analysis
- Approach explanation

The goal is to track progress, learn different algorithms and data structures, and provide reference implementations.

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/jorgeadev/leetcode-answers.git
   cd leetcode-answers
   ```

2. Navigate to the desired language and difficulty folder

3. Each problem folder contains:
   - Solution file(s)
   - README.md with problem description and analysis
